# ❄ Pankaj And Deepak Ice Factory Website

This is a simple web-based ordering system for an ice factory built using HTML, CSS, and JavaScript.

## 🧊 Features
- Customer order system (25KG & 50KG ice bags)
- Admin panel with PIN login
- Editable price system
- Order management (edit/delete/status)
- Delivery/Pickup date selection
- Pay at pickup point system
- Local storage database (no backend required)
